TraduCrossing (anteriormente chamado de ACBR) é um projeto focado em traduzir o jogo Animal Crossing do GameCube para Português (Brasil).

**Tenha em mente que por ser um projeto de uma pessoa só, estou usando ferramentas de IA para agilizar o processo de tradução, mas aplicando correções textuais ou diálogos fora de contexto.**

Tradução testada no emulador Dolphin, Nintendo Wii, GameCube, e nos ports para PC e PS Vita.


**INSTRUÇÔES:**

(Requer o GameCube File Tools https://github.com/LagoLunatic/GCFT)

1. Importe o arquivo ISO ou GCM do jogo;
2. Troque os arquivos "forest_1st.arc" e "forest_2nd.arc" originais pelos baixados, clicando em "Replace File" com o mouse direito;
3. Clique em "Export GCM" e salve como ISO ou GCM
